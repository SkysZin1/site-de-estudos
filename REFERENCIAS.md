# Referências internas

A pasta `referencias/` será usada apenas como base interna de consulta do projeto.

Os PDFs adicionados nela servirão para criar resumos, fórmulas, exemplos práticos e exercícios para as páginas do site. Esses materiais não precisam ser citados diretamente nas páginas públicas do site, pois funcionam como apoio de estudo e organização.

## Organização das referências

- `referencias/FOE/`: arquivos de FOE.
- `referencias/FT/`: arquivos de Fenômenos de Transporte.
- `referencias/Calculo3-MMF/`: arquivos de Cálculo 3 / Modelagem Matemática de Fenômenos.

Mantenha os arquivos nomeados de forma clara para facilitar a consulta futura.
