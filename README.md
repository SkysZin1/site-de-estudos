# Site de Estudos

Projeto inicial de um site de estudos para organizar matérias, resumos, fórmulas, exemplos e exercícios.

## Objetivo

Criar uma base simples e expansível para consulta acadêmica, com páginas separadas para:

- FOE - Fenômenos Ondulatórios e Eletromagnéticos.
- FT - Fenômenos de Transporte.
- Cálculo 3 - Modelagem Matemática de Fenômenos.

## Tecnologias usadas

- HTML5.
- CSS3.
- JavaScript simples.
- Git.

## Estrutura de pastas

```text
site-de-estudos/
├── index.html
├── README.md
├── REFERENCIAS.md
├── materias/
│   ├── foe.html
│   ├── ft.html
│   └── calculo3.html
├── assets/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── main.js
└── referencias/
    ├── FOE/
    │   └── .gitkeep
    ├── FT/
    │   └── .gitkeep
    └── Calculo3-MMF/
        └── .gitkeep
```

## Como abrir localmente

Abra o arquivo `index.html` diretamente no navegador.

Também é possível abrir a pasta do projeto em um editor de código e iniciar pelo arquivo:

```text
site-de-estudos/index.html
```

## Como adicionar PDFs de referência

Adicione os PDFs de estudo dentro da pasta `referencias/`, separados por matéria:

- `referencias/FOE/`: arquivos de FOE.
- `referencias/FT/`: arquivos de Fenômenos de Transporte.
- `referencias/Calculo3-MMF/`: arquivos de Cálculo 3 / Modelagem Matemática de Fenômenos.

Esses arquivos servem como base interna de consulta para criar resumos, fórmulas, exemplos e exercícios.
